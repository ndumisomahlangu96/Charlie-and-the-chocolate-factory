class selector

The id is usually unique for an element. But what if we need the same CSS
for a group of elements? Not all, but some of them. Observe the following

Out of all the six paragraphs, first, three should be red in color. We can give
the same ids to them and it will work. But it is not recommended at all. For
such cases, we have another attribute, which is known as the "class"
attribute.

The class selector is used to give the same CSS to multiple elements. To
use this type of selector, use a dot(.) followed by the class name.
The same class is assigned for the first three tags.
