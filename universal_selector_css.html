Universal selector

So far, we have discussed how to use the id selector for a particular element,
the class selector for multiple elements, and the grouping selector for the
grouping different elements with the same definition. But what if we need the
same CSS for every element in the HTML file? There is another selector
called universal selector for such cases. Use the asterisk (*) sign to apply CSS to every element.
