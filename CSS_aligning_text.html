Aligning text

By default, the text is always aligned to the left side of the container. But
can also change the alignment using CSS.

The "text-align" property sets the horisontal alignment. It can have four
values: left, right, and center.

The first paragraph is aligned to the left, which is also the default alignment,
the second paragraph is aligned to the right, and the third one is aligned to the
center.
