Observe the following HTML code.

This is a red paragraph.

This is a blue paragraph

This is a green paragraph

There are three paragrapsh but no CSS is applied to them. We have to give
different colors to each paragraph. How can we do it? We need to select a
particular paragraph and apply a particular color to it. To do this, the id
selector is used.

Every element has an "id" attribute. Id given to an element should be unique.

Each paragraph has a unique id and unique CSS can be applied to each of
them using this id.

To select an element using the id, use the hash(#) sign followed by the id
name.
