Text decoration

Generally, text in HTML does not have any decorations. To add decorations
or to remove the default ones, use the "text-decoration" property.

The "text-decoration" property can have four values: underline, line-
through, overline, and none. The first three values add decorations while
the last one removes the default decoration.
